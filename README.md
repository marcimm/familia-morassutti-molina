# História da Família — Site Astro

Este projeto é um esqueleto de site estático feito com [Astro](https://astro.build),
pensado para abrigar o conteúdo do livro da história da família (Morassutti–Molina e ramos relacionados).

## Como usar

1. Instale as dependências:

   ```bash
   npm install
   ```

2. Rode o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```

3. Abra o navegador em `http://localhost:4321`.

Depois, basta editar os arquivos em `src/pages/` e ir colando o texto dos capítulos,
ajustando os títulos e seções conforme a sua narrativa.
